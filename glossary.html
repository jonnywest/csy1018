Glossary

Wednesday, 02 January 2019

HTML Terms

Below is a list of some basic html elements that can be used to build a web page, a full and concise glossary can be found at w3schools

HTML - HTML stands for Hyper Text Markup Language. HTML is the language used to create Web pages for display in Web Browsers.

HTML tag - he html tag tells the browser that this is an HTML document.

!DOCTYPE - The !DOCTYPE declaration must be the very first thing in your HTML document, before the html tag. The !DOCTYPE declaration is not an HTML tag; it is an instruction to the web browser about what version of HTML the page is written in.

h1-h6 - The h1 to h6 tags are used to define HTML headings The h1 defines the most important heading. h6 defines the least important heading.

Head tag - The head element is a container for all the head elements. The head element can include a title for the document, scripts, styles, meta information, and more.

Body tag - The body tag defines the document's body.The body element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

Nav tag - The nav tag defines a set of navigation links.

CSS Terms

Below is a list of some basic css properties that can be used to style a web page, a full and concise glossary can be found at w3schools

Background - The background property can be used to set the values of the background for multiple things such as color, size, position ect.

Align-content - The align-content property modifies the behavior of the flex-wrap property. It is similar to align-items, but instead of aligning flex items, it aligns flex lines.

Border - The border property is used to set the style of the borders for different elements

Color - Set the text-color for different elements

Font family - The font-family property specifies the font for an element. The font-family property can hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font.

@media - The @media rule is used in media queries to apply different styles for different media types/devices. Media queries can be used to check many things, such as: width and height of the viewport width and height of the device orientation (is the tablet/phone in landscape or portrait mode?) resolution
